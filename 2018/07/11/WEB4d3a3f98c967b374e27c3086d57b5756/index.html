    2018-07-11
    Code

    • crx下载地址
      链接: https://pan.baidu.com/s/1slLL7JZ 密码: 75ws

    • 安装直接将crx拖拽到 chrome://extensions/ 里

    • 重启chrome，F12看到react面板
