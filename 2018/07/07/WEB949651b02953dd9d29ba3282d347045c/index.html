    代码规范

    2018-07-07
    Code
    • 二元运算符左右两边加空格
      • if, for 和括号之间加空格
      • 严格按照要求进行程序缩进
        即使 if / for
      • 语句内部只有一句话，也要加上花括号
      • 变量名使用有意义的英文名，不要用a,b,c,s1,s2
      • 区分不同的逻辑块，逻辑块之间用空行隔开，简要注释每个部分做的事情
      • 多用 Helper Function 或子函数，不要所有程序都写在一个大函数里
